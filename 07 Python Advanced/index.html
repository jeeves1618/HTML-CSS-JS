5. Lists

Lists behave in the same way of JS array. list=[1,2,ball, 3.7, [5,6,7]] is an example of a python. You can get the length of list by using len(list). You can access the link using indexes. As it was with arrays list[0] will return 1 and list[-1] will return [5,6,7]. List can be sliced (list[1:], list[:2], list[2,2] etc.). Unlike Strings lists are mutable. list[0]=7 is ok.

Consider my_list = ['a','b','c']. If you do my_list.append(['c','d']) will result in ['a','b','c',['c','d']]. But, my_list.extend(['c','d']) will result in ['a','b','c','c','d']. temp = my_list.pop() will pop the last item and store it in temp. You can remove the specific item by providing index as parameter in the pop menthod.my_list.reverse will result in ['c','b','a']

7. List Operations

This segment, my_list.sort(), will sort the my_list and store the output in the same variable, my_list. Let n_list=[1,2,[7,8] be a nested list. n_list[2] will give [7,8] and n_list[2][1] will return 8. List comprehension is another powerful feature in Python. 8. Dictionaries

Dictionaries are python's version of hash table. In Javascript we called the similar key value pairs as objects. Unlike lists, dictionaries do not retain any order and are accessed through their keys. An example of dictionary is my_detail = {"fname":"Jane", 'lname':"smith",'fav_nums':[5,7,8]}. To access 7, a element in a list nested in dictionary, we write my_detail['fav_nums'][1]

9. Manipulate dictionary

To update a value in the dictionary, just use the key. my_detail['fname'] ='John'. If you want to add a new key value pair, use a new key my_detail ['age'] = 38. my_detail.popitem() will remove the last item in the dictionary. my_detail.pop('age') will remove the age. del my_detail['fname'] will delete the element whereas del my_detail will delete the dictionary. my_detail.clear() just empties it.

10. Tuples and sets

Tuples are immutable sequences. They are lists that cannot be mutated like list(0)=7. Tuples are created in the same way as lists by replacing [] with (). t = (0,1,3) is a tuple. They are accessed in the same way as lists using []. Sets are unordered collection of unique elements. s = set() s.add(1) s.add(4) is how a set is constructed. if you do s.add(4) again, set will remain as {1,4}
