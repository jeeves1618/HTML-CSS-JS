HTML

  • First Things...

    HTML is a mark up language. What does a mark up language mean? It means HTML is used for annotating the content in a web page. A web page neads headers, sub headers, links etc. to present the content in a meaningful form to its consumers. The consumers of a web page can be humans or other programs.

    In order to mark up the content, HTML uses tags. The tags that clearly describes their intent to users, browsers and programs without relying on class or id selectors are called semantic elements. For example, <article> or <main> are all semantic tags. Non semantic tags are generic in nature like, <div> or <p>, and will need a class="main" to describe itself. Tags and content put together is called an element.

    A browser will always rended the HTML from top to bottom.
