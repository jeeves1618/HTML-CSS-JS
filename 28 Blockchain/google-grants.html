Google Authorization Server

  • Google Photos API

    Lets go to https://developers.google.com/photos and navigate to Get started with REST. Create a new project with no organization.

    Under that project, search for Photos Library API and enable it.

  • Config and Credentials

    Once the API is enabled, configure the OAuth Consent Screen. Select the user type as external and add your id as test user ID. Rest of the fields can be defaulted or left alone for now.

    Then proceed to the creation of credentials. Choose OAuth Client ID and the application type as Web Application.
