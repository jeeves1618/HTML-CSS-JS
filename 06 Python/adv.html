2. Global

x = 50
def func():
    global x;
    x = 100;
print("Before the func, x is ",x)
func()
print("After the func, x is ",x)
 x would have been 50 second time if global wasn't used. Now the output is
Before the func, x is 50
After the func, x is 100

4. Class with Method

class Circle(object):
   pi = 22/7
   def __init__(self, r=10):
       self.r = r
   def area(self):
       return self.r**2*Circle.pi
c1 = Circle()
print(c1.area())
c2 = Circle(r=20) will override the default radius of 10. pi is a static variable and is a property of class here.

8. Reg Expressions 101

import re
words=['cat','dog']
text='Tom is a famous cat'
for word in words:
   if re.search(word,text):
      print('Searching ' + word + ' :found')
   else:
      print('Searching ' + word + ' :not found'). O/P: Searching cat :found & Searching dog :not found

9. Reg Expressions 201

re.split(splitter,text) will split the text at the splitter. type(re.search (word,text)) will return <class 're.Match'>. This class has many in built functions to leverage. For instance, re.search(word, text).start() gives the index at which the word starts. re.findall('a',text) returns a list ['a', 'a', 'a']. Repetition syntax can be used instead of 'a'

11. Returning Functions

      print('Hello Mam')

      print('Hello Sir')

      return greetmam
   else:
      return greetsir

x() #invokes and prints Hello Mam

12. Passing functions

def calling(func):
   def insider():
      print('before func')
      func()
      print('after func')
   return insider
def called():
   print('its twisted')
called = calling(called)
called(). And the o/p will be before func, its twisted and after func
