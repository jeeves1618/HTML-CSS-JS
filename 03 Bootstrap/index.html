Bootstrap

What is Bootstrap?

Bootstrap is the most popular HTML, CSS and JS framework for developing responsive mobile first projects on the web. Bootstrap consists of mostly CSS. And it expects you to have a specific HTML structure. Nothing complex, it just wants to have some div elements here and there. It is also called mobile first framework.

One complaint against Bootstrap is that it is bloated. But, remember a lot of features in Bootstrap will never be used by us. So, it still makes sense to download or CDN selectively rather writing the code on our own.

Add your CSS style after the bootstrap CDN. In this way, you will be able to override the bootstrap styles from your style sheet.

Bootstrap Grid System

The Bootstrap grid should always be inside a container. This can be acvhieved using a container class or a container fluid class. <div class="container">

A class called row will be used to create horizontal groups of columns.
