What is ML?

  • Common ML is supervised learning (manual labeling). For instance spam filtering, speech recognition etc.
  • The performance of supervised learning is directly proportional to amount of available data and the size of nueral networks.
  • ML looks at the old inputs and outputs and predicts outputs for new inputs. Data science gleans new insights out of those inputs and outputs.
  • Ml is the field of study that gives computers the ability to learn without being explicitly programmed. DS is the science of extracting knowledge and insights from data.
  • The output of ML is a program that computes output for an input. The output of DS can very well be a data science.
  • Deep learning or artificial neural network is nothing but a big mathematical equation that gives an output for a bunch of inputs.

What is an AI company?

  • Lets try answering a more basic question. Is a shopping mall with a website, an internet company? No. Because you can't do A/B testing with shopping mall. There is no short iteration time. In internet companies, decision making is pushed down to engineers and other specialized roles.
  • In the same way, any company with a deep learning project is not an AI company. An AI company is very good at strategic data acquisition. They have unified data warehouse. Very good at automation. They have roles like MLE.
  • How can a company become good at AI?
    1. Execute pilot projects to gain momentum.
    2. Build an in-house AI team.
    3. Provide broad AI training.
    4. Develop an AI strategy.
    5. Develop internal and external communications.

Workflow of Machine Learning Project

  • Collect Data
  • Train model using supervised learning iterations
  • Deploy model
  • Get data back from users and keep tuning the model

Workflow of Data Science Project

  • Collect Data
  • Analyze Data. Iterate many times to get good insights
  • Come up with hyppthesis and suggest actions
  • Deploy changes
  • Re-analyze new data periodically

How does a smart speaker work?

  • Wakeword Detection. Something like "Hey Siri" or "Alexa"
  • Speech Recognition. Takes the sound bytes after wakeword and transcripts them into a text.
  • Intent Recognition. Look for intent like "Music", "Call", "Netflix" etc. A typical speaker will have limited number of words associated with intents.
  • Intent Execution. A software program tranlates the intent into action.

Major AI Application Areas - Computer Vision

  • Image classification or Object recognition, where the image is taken as a whole.
  • Object detection where the AI tries to identify objects within an image. The output can be rectangles around the cars, kerbs or other obstacles scanned by a self driving car camera.
  • Image segmentation. Same as object detection, but instead of going for rectangles, it goes for pixel level segmentation. Reading an X Ray would require image segmentation.
  • Tracking. Ability to follow moving objects is called tracking.

Major AI Application Areas - NLP

  • Text classification - Spam email or not, product categorization etc. Sentiment recognition is advanced text classification.
  • Information retrieval
  • Name entity recognition - Finding people names, location names from text.
  • Machine Translation - Japanese to English etc.
  • Work on communicating the successes
