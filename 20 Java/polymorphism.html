Polymorphism

  • Composition

    Inheritance defines an IS A relationship. Composition defines an HAS A relationship.

    So, it is a challenge if we want to manipulate data passed as parameter before calling super. We can try something like this.

    super(isTurboCharged,engineVolumeInCC < 1000 ? "Small" : (engineVolumeInCC > 1500 ? "Large" : "Medium"))

    If a subclass doesn't make a call to super(), java will make that call to the default constructor of the super class. If the default constructor of the super class isn't there, because a constructor was explicitly defined, then the subclass must explicitly call that constructor in the super class.

    A object of type sub class can be passed to the method accepting the super class. But, the converse is not possible.
