Heroku

  • Signing up

    Heroku supports the deployment of software developed in the NodeJS, Java, Python, PHP, Go, Ruby, Scala and Clojure technology stacks. Go to https://www.heroku.com/. You should be able to sign up using your private email.

    Once you sign up, you will end up in the app dashboard. There you have to give an unique name to the app.
